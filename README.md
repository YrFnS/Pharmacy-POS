# Pharmacy POS System

A comprehensive Point of Sale (POS) and inventory management system built specifically for pharmacies and drug stores.

## Features

- **Point of Sale (POS)**
  - Fast and intuitive one-page checkout screen.
  - Smart search by barcode, brand name, or generic name.
  - Drug substitutes recommendations for out-of-stock items.
  - Cart management with quantity adjustments, line-item discounts, and returns.
  - Place transactions on hold and resume them later.

- **Payment Processing**
  - Support for multiple payment methods: Cash, Credit Card, and Store Credit (Tab).
  - Quick cash preset buttons and automatic change calculation.

- **Shift Management**
  - Open shifts with starting cash drawer amounts.
  - Close shifts and generate end-of-day Z-Reports.

- **Inventory Management**
  - Track products by category, manufacturer, and shelf location.
  - Deep batch tracking (track individual batches, quantities, and their expiry dates).
  - Receive newly procured stock seamlessly.

- **Alerts & Dashboard**
  - Visual dashboard for pharmacy managers.
  - Alerts for items expiring soon.
  - Alerts for low stock (below minimum threshold).

- **Customer / Patient Directory**
  - Track patient purchase history.
  - Manage store credit, tabs, and debt tracking.

- **Settings & Localization**
  - Built-in multi-language and RTL (Right-to-Left) support (English, Arabic, Kurdish).
  - Clean, minimal, and performant UI.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Icons:** Lucide React
